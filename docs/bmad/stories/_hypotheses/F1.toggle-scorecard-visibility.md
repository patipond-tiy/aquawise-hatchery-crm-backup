## Story

**As** an Owner, **I want to** flip the public scorecard on/off and pick which stats appear, **so that** I control what customers see on my public nursery page.

## Acceptance Criteria

> All AC below are planning-grade hypotheses. Toggles exist in the UI today but the public scorecard they control (F2) does not exist and must not be built until the F-epic is validated.

- Six toggles: `public`, `showD30`, `showPCR`, `showRetention`, `showVolume`, `showReviews`
- Changes are optimistic with rollback on error
- `showReviews` is disabled (read-only / grayed-out) until a reviews schema ships — the toggle column exists but the feature has no data source
- Persist to `scorecard_settings` row scoped to the calling user's `nursery_id`
- Only `owner` role can write; `auditor` and `counter_staff` see the page read-only
